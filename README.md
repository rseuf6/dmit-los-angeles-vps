## 常见问题

### DMIT洛杉矶VPS适合建站吗？

适合，但要看建站类型。如果目标访客主要在国内，Pro系列的CN2 GIA回程优化对国内访问速度有明显帮助。如果目标访客在北美或全球，EB系列性价比更高，没必要为回国线路付溢价。我自己有一个面向国内读者的小站跑在Pro Starter上，首屏加载在2秒以内，体验可以接受。

### DMIT有没有退款保证？

DMIT提供3天退款政策（部分套餐），具体条款以购买时官网说明为准。建议在下单前确认当前套餐的退款条件，避免误解。

### 洛杉矶CN2 GIA和普通线路延迟差多少？

从国内主流城市到洛杉矶，CN2 GIA高峰期延迟通常在160ms–190ms，普通线路高峰期可能飙到250ms甚至更高，且丢包率明显上升。如果你的应用对实时性有要求（比如视频会议、实时数据同步），这个差距会直接影响体验。

### DMIT支持Windows系统吗？

DMIT洛杉矶VPS主要提供Linux系统镜像（Debian、Ubuntu、CentOS等），部分套餐支持自定义ISO安装Windows，但需要自备授权。具体支持情况以购买页面为准。

### 便宜的Lite套餐和EB套餐有什么本质区别？

Lite的核心限制是带宽上限（200Mbps），EB则是1Gbps共享带宽。实际使用中，Lite在低峰期速度不差，但高峰期带宽争抢更明显。EB的网络线路质量比Lite高一档，回程有优化，Lite则相对基础。预算极限选Lite，有一点余量选EB。
